# Clicky Attribution

This repository began from the open-source Clicky macOS app by Farza and is now being reshaped into Local Jarvis.

The current app direction is documented in `README.md`: a macOS menu bar Jarvis that sees the screen, uses GPT-5.5 through a Cloudflare Worker, and operates apps with macOS mouse and keyboard tools.
